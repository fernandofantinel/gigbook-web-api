GigBook is a modern application for musicians to manage and display song lyrics and setlists during live performances, rehearsals, and practice sessions.

## 🚀 Features

- Full setlist display
- Optimized dark mode for stage use
- Large, readable typography
- Song navigation (previous/next)
- Search songs and past setlists
- Support for extra songs during performances

## 🧱 Tech Stack

- Next.js (App Router)
- React Native (Expo)
- TypeScript
- Prisma + PostgreSQL (Neon)
- Tailwind CSS
- Vitest
- ESLint + Prettier

## 📦 Project Structure (planned)

- Web application (management + API)
- Mobile application (lyrics display)
